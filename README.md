# Project Overview 
* Project Name : urlparser
* Project Base : Springboot, Java8, Maven 
* Usecase : 입력된 url의 컨텐츠 문자열을 숫자/영문을 구분하여 필터링/추출/정렬/병합/그룹핑 하여 결과 출력 
  

# Page Example 
* 실행 방법 : Project Load 후 UrlParserApplication 실행 
* 실행 경로 : http://localhost:8080
